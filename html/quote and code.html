JavaScript 변수 선언

변수란?

변수는 데이터를 저장하는 공간입니다. JavaScript에서는 var, let, const 키워드를 사용합니다.

기본 사용법

// 변수 선언과 할당
let userName = "김개발";
const age = 20;
var isStudent = true;

// 변수 사용
console.log("이름: " + userName);
console.log("나이: " + age);

전문가 조언

현대 JavaScript에서는 let과 const를 사용하고, var는 피하는 것이 좋습니다. 이는 스코프 문제를 방지하고 더 안전한 코드를 작성할 수 있게 해줍니다.

- JavaScript 개발 가이드라인

실습 예제

다음 코드를 실행해보세요:

let message = "Hello, World!";
console.log(message);

브라우저 개발자 도구에서 Hello, World!가 출력되는 것을 확인할 수 있습니다.
